## 🎯 Learning Outcomes

- Declare and use variables (`var`, `let`, `const`)
- Work with JavaScript data types and type coercion
- Apply arithmetic, comparison, and logical operators
- Implement conditional logic using `if/else` and `switch`
- Use loop constructs (`for`, `while`, `do-while`)
- Define and invoke functions with proper scope understanding
