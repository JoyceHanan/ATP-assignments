## 🌟 Highlight Project — Employee Management System

One of the major projects developed during this program is the **Employee Management System**, created as a complete MERN stack application.

### 🔹 Key Features
- Employee record management
- Full CRUD functionality
- Dynamic frontend with React
- RESTful backend APIs
- MongoDB database integration
- Client-side routing using React Router

### 🔹 Tech Used
- **Frontend:** React + Vite
- **Backend:** Node.js + Express.js
- **Database:** MongoDB + Mongoose

This project helped strengthen my understanding of scalable application structure and frontend-backend communication.
